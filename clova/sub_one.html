    는 음성 합성 기술 연구를 통해 사람처럼 자

    Demo
  • CLOVA OCR 광학 문자 판독

    의 글자를 판독하는 독자적인 텍스트 영역 검
    출법을 보유하고 있습니다.

    Demo

  • CLOVA Chatbot 챗봇

    자연어 처리 기반으로 최신 머신러닝 기술을
    활용해 입력된 문장의 형태소와 개체명, 구문
    을 분석, 의미를 판별하고, 사용자의 질의에
    대한 최적의 답변을 제공합니다.

  • CLOVA Vision 이미지 분석

    네이버, 라인의 방대한 데이터를 기반으로 선
    행 학습된 모델을 통해 이미지에서 유용한 정
    보를 추출하거나, 필요한 이미지를 검색, 형태 변형 들의 기능을 제공합니다.

  • CLOVA Text Analytics 텍스트 분석

    다량의 비구조적 데이터를 구조화된 데이터
    로 변환하여 분석할 수 있습니다. 감정분석,
    키워드 추출, 문서 요약 등의 기능을 통해 효
    율적으로 데이터 분석을 할 수 있습니다.

  • CLOVA Face 얼굴 분석

    이미지, 영상에서 얼굴이 있는 위치를 파악하
    고, 분석할 수 있습니다. 그리고 식별된 얼굴
    에서 성별, 나이, 감정, 표정 등 복잡하고 다양
    한 특징 정보를 제공합니다.

  • CLOVA Video 동영상

    영상 스트리밍 환경이나 저장된 동영상에서
    인물, 물체, 장소, 장면 등 다양한 객체를 자동
    으로 인식하여 검색, 편집, 추천 등의 기능을
    제공합니다.

  • CLOVA NSML 머신러닝 플랫폼

    딥 러닝 알고리즘 설계 및 개발을 돕는 플랫
    폼으로서 자동 GPU 스케줄링과 AutoML 기
    능을 제공합니다. 리더보드로 협업하고, 딥
    러닝 모델을 API 형식으로 제공합니다.
